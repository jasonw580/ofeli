Detailed Description

This class implements a singly linked list used by the implementation of Shi and Karl' algorithm.

The elements stored are integers that correspond of the offset of the level set function buffer.

Examples:
interface.cpp, and linked_list.cpp.

int ofeli::List::size ( ) const

Returns the number of elements in the list without counting the sentinel node.

The complexity of this function is in O(n) so if you want to get often the size of the list, you should update the size in a variable after each modification.
